### HTML

简介：HTML是超文本标记语言（HyperText Markup Language）的缩写。HTML 是构成 Web 世界的一砖一瓦。它定义了网页内容的含义和结构。HTML 不是一门编程语言，而是一种用于定义内容结构的标记语言。

#### HTML文档结构
```markdown
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
  <title>【网页标题】</title>
</head>
<body>
  <h1>【页面内容题目】</h1>
  <p>【文本内容】</p>
</body>
</html>
```
